I. Answer the following shortcuts and their purposes

  1. Ctrl + N: Creates a new document.
  2. Ctrl + C: Copies the selected text or item to the clipboard.
  3. Ctrl + Z: Undoes the last action.
  4. Ctrl + Shift + >: Increases the font size.
  5. Ctrl + E: Aligns text to the center.

II. Differentiate between the following pairs of shortcuts. Explain their usage:

  1. Ctrl + X and Ctrl + C:
    • Ctrl + X: Cuts the selected text or item, removing it from its current location and placing it in the clipboard.
    • Ctrl + C: Copies the selected text or item to the clipboard without removing it from its original location.
  2. Ctrl + E and Ctrl + L:
    • Ctrl + E: Aligns the selected text to the center.
    • Ctrl + L: Aligns the selected text to the left.
  3. Ctrl + Shift + > and Ctrl + Shift + <:
    • Ctrl + Shift + >: Increases the font size of the selected text.
    • Ctrl + Shift + <: Decreases the font size of the selected text.
  4. Ctrl + Z and Ctrl + Y:
    • Ctrl + Z: Undoes the last action performed (reverts changes made).
    • Ctrl + Y: Redoes the last action that was undone (restores the previous change).

IV. Answer the following questions.

1. What is a Word Document, and why do we use it?

A Word Document is used to create, edit, format, and share text-based content like letters, reports, resumes, or essays.

Why Do We Use It?

  • Ease of Editing: Allows for easy changes in text, font, and layout.
  • Professional Appearance: Offers tools to format content in a visually appealing way.
  • Collaboration: Enables sharing and editing with others.
  • Versatility: Can include images, tables, charts, and hyperlinks alongside text.
  • Storage and Sharing: Files can be saved digitally and shared easily through email or cloud storage.

2. What are the types of alignment, and why are they necessary?

  • Left Alignment: Text is aligned to the left margin.
  • Right Alignment: Text is aligned to the right margin.
  • Center Alignment: Text is centered between the left and right margins.
  • Justify Alignment: Text is spaced so that it aligns evenly on both left and right margins.

3. Why Do We Use Shapes in Word?

  • Visual Representation: Shapes like circles, arrows, and rectangles help present information in a visually appealing way.
  • Highlighting Content: Shapes can emphasize important points, such as key headings or sections.
  • Creating Diagrams: Use shapes to design flowcharts, mind maps, or organizational charts.
  • Improving Engagement: Adding shapes makes documents more engaging and interactive, especially for presentations or reports.
  • Customization: Shapes can be resized, colored, and styled to match the document’s theme or purpose.
